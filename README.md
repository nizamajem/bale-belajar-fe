# BaleBelajar FE

Frontend utama BaleBelajar menggunakan Next.js App Router, TypeScript strict, Tailwind CSS, Framer Motion, dan Lucide Icons.

## Menjalankan

```bash
npm install
npm run dev
```

Aplikasi berjalan di `http://localhost:3000`.

## Environment

Salin `.env.example` menjadi `.env.local` bila perlu.
